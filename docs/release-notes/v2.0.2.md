# Bu课表 v2.0.2

## 新增功能

- 新增学期管理：支持创建、切换、删除学期，课程数据按学期独立管理。
- 新增学期导入/导出：导入时可选择目标学期，也可将指定学期导出为 CSV。
- 新增 ICS 日历导入：支持从 `.ics` 文件导入课程。
- 新增大小桌面小组件：可在桌面查看今日/明日课程概览。
- 新增周视图课程卡片密度设置：支持紧凑、标准、舒展三种排版，字号变化会配套调整信息量和间距。
- 关于页面更新为 v2.0.2，点击可跳转到 GitHub 项目。

## 修复与优化

- 修复点击桌面小组件不跳转应用的问题，大小组件和小组件课程列表项均可打开应用。
- 优化依赖体积，移除未使用的 Compose tooling/扩展图标依赖，减小安装包体积。
- 调整部分图标依赖到 Compose core icons，保留原有功能入口。
- 为课程仓库的协程流操作补充显式 opt-in，减少编译警告。

## 安装说明

- 本次提供 `app-release-v2.0.2-self-signed.apk` 自签名安装包。
- 如果手机上已安装使用不同签名的旧版本，可能需要先卸载旧包再安装。
- 后续覆盖安装同一自签名版本线时，需要继续使用同一份本地签名 key。
